## 📝 技術仕様

### データ構造
```json
{
  "version_info": {
    "scarlet_violet": {
      "availability": "scarlet|violet|both|none",
      "notes": "補足情報"
    },
    "sword_shield": {
      "availability": "sword|shield|both|none", 
      "notes": "補足情報"
    }
  }
}
```

### フィルター値
- `all`: 全て表示
- `scarlet`/`violet`: バージョン限定
- `sword`/`shield`: バージョン限定
- `both`: 共通ポケモン
- `none`: 対象外
