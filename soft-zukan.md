# ソフト図鑑とは

ポケモンにおけるソフト図鑑とは、**Pokémon HOMEに登録される各ソフトごとの図鑑**のことを指す。

『ポケモン スカーレット・バイオレット』のパルデア図鑑や『ポケモン ソード・シールド』のガラル図鑑など、『ポケットモンスター』シリーズ各ソフトの図鑑が確認できる。

## 登録の仕組み

例えば、『ポケモン ソード・シールド』で捕まえたイーブイを『Pokémon HOME』に預けると:

- ✓ 『Pokémon HOME』の「全国図鑑」に登録
- ✓ ソフト図鑑の「ガラル図鑑」に登録
- ✗ 「シンオウ図鑑」など、他のソフト図鑑には**登録されない**

「シンオウ図鑑」のイーブイを登録するには、『ポケモン ブリリアントダイヤモンド・シャイニングパール』で捕まえたイーブイを『Pokémon HOME』に預ける必要がある。

**つまり、Pokémon HOMEにすべての種類のポケモンを揃えるだけでは完成せず、各ソフトでポケモンを捕まえてPokémon HOMEへ転送しなければならない。**

## 産地マークの役割

ポケモンには産地マークが導入されており、**そのソフトでゲットしたポケモンを記録している**。この機能を使ってソフト図鑑に登録される。

### 具体例:ガラル御三家の場合

スカーレット・バイオレットにおいて、ガラル御三家をソード・シールドから連れてきた場合:

1. **ガラルマークのまま**
   - ゲーム内のブルーベリー図鑑には登録される
   - HOMEへ送っても**スカーレット・バイオレットのソフト図鑑には反映されない**

2. **タマゴを孵化させると**
   - スカーレット・バイオレット内で孵化させると**パルデアマーク**がつく
   - HOMEへ送ると**スカーレット・バイオレットのソフト図鑑に登録される**

3. **補足:進化させる場合**
   - 生まれたポケモンをソード・シールドへ送って進化させても**パルデアマークは変わらない**
   - 進化後にHOMEへ送れば**スカーレット・バイオレットのソフト図鑑に登録される**

## DLC図鑑と重複登録

スカーレット・バイオレットやソード・シールドにはDLCがあり、1つのソフトに複数の地方図鑑が存在する。

| ソフト | 図鑑 |
|--------|------|
| スカーレット・バイオレット | パルデア図鑑 / キタカミ図鑑 / ブルーベリー図鑑 |
| ソード・シールド | ガラル図鑑 / ヨロイ島図鑑 / カンムリ雪原図鑑 |

### 重複登録のルール

**同じソフト内の複数図鑑に登場するポケモンは、1回の捕獲で全図鑑に同時登録される。**

#### 具体例: コンパン・モルフォン（SV）

コンパンはSVのパルデア図鑑・キタカミ図鑑・ブルーベリー図鑑の3つすべてに登場する。

- SVでコンパンを1匹捕まえてHOMEに送ると:
  - ✓ パルデア図鑑に登録
  - ✓ キタカミ図鑑に登録
  - ✓ ブルーベリー図鑑に登録
- つまり **1回の捕獲で3つの図鑑枠が同時に埋まる**

### ソフト図鑑完成に必要な実際の捕獲数

各ソフトの図鑑完成に必要な捕獲数は「ユニークポケモン数」であり、図鑑の合計枠数ではない。

| ソフト | 図鑑合計枠数 | 重複 | ユニーク数（実際の捕獲数） |
|--------|-------------|------|--------------------------|
| スカーレット・バイオレット | 669 | 180 | 489 |
| ソード・シールド | 642 | 231 | 411 |

## ソフト図鑑の対象ソフト

ソフト図鑑はNintendo Switch以降のソフトが対象。3DSソフト（XY、ORAS、SM、USUM）にはソフト図鑑の概念がない。

| プラットフォーム | ソフト | ソフト図鑑 |
|----------------|--------|-----------|
| Switch | スカーレット・バイオレット | ✓ 対象 |
| Switch | ソード・シールド | ✓ 対象 |
| 3DS | ウルトラサン・ウルトラムーン | ✗ 非対象 |
| 3DS | サン・ムーン | ✗ 非対象 |
| 3DS | オメガルビー・アルファサファイア | ✗ 非対象 |
| 3DS | X・Y | ✗ 非対象 |

3DSソフトのデータは「どのゲームで捕まえられるか」の参考情報として活用する。
